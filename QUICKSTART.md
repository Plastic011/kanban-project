# ⚡ Quick Start (2 Minutes)

## Installation

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Open: **http://localhost:5173**

## Features

✅ Drag & drop tasks  
✅ Search & filter  
✅ Task editing  
✅ Team assignments  
✅ Time tracking  
✅ Modern dark UI  

## Build for Production

```bash
npm run build
```

Creates optimized `dist/` folder.
